远程操作

Pycharm 简单操作

  1. 通过git clone命令同步到本地以后，使用Pycharm进行编辑。
  2. 工作完成后，点击顶部的VCS或者快捷键ctrl + K 进行文件提交。提交时可点击右下角commit旁边的下拉菜单选择commit and push，直接进行提交和推送操作。
  3. 修改后的文件推送后，在码云项目中点击pull requests将修改后的代码提交给远程库。
  4. 开始编辑前同步远程库

同步远程库

  • 给fork配置一个 remote
  • 使用git remote -v查看远程状态
  • 添加一个将被同步给 fork 远程的上游仓库
  • 再次查看状态确认是否配置成功。

同步fork

  • 从上游仓库 fetch 分支和提交点，提交给本地 master，并会被存储在一个本地分支 upstream/master
  • 切换到本地主分支（如果不在的话）
  • 把 upstream/master 分支合并到本地 master 上，这样就完成了同步，并且不会丢掉本地修改的内容。

*如果想更新到 GitHub 的 fork 上，直接 git push origin master 就好了。
